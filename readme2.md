# HR Dashboard

Внутренняя HR-система для управления сотрудниками и просмотра аналитики.

## Технологии

- React 18 + TypeScript
- Redux Toolkit + RTK Query (кэширование запросов)
- MUI (Material UI)
- React Hook Form
- ECharts
- Vite

## Функциональность

- Таблица сотрудников с фильтрацией по отделу
- Добавление и удаление сотрудников
- Валидация форм
- Страница аналитики с графиками (круговая диаграмма, столбчатый график)
- Кэширование запросов через RTK Query

## Что реализовано из оптимизаций

- **RTK Query** — кэширование данных при переходе между страницами
- **useMemo** — мемоизация вычислений для фильтрации и графиков
- **useCallback** — стабильные ссылки на функции для предотвращения лишних ререндеров
- **React.memo** — предотвращение ререндера строк таблицы при неизменных пропсах
- **ECharts** — Canvas рендеринг для производительной отрисовки графиков

## Запуск
```bash
# Установка зависимостей
npm install

# Запуск бэкенда (порт 3001)
npm run server

# Запуск приложения (порт 5173)
npm run dev
```
